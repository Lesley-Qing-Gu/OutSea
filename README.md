# 🌊 离岸 OutSea

> 「离岸，是在异地连线，也是在银幕中寻找坐标。」

《离岸 OutSea》是一档由两位身在欧陆的影迷发起的播客。我们分别在斯德哥尔摩和阿姆斯特丹，通过简单的设备跨越海域，记录那些发生在电影节现场、古老资料馆和街角艺术院里的光影故事。

这是离岸播客的官方网站。

## 主创

- **Lesley** / Stockholm
- **Tiki** / Amsterdam

## 栏目

| 栏目 | 英文 | 简介 |
|------|------|------|
| 影节前线 | Film Festival | 深度直击欧洲三大电影节（戛纳/威尼斯/柏林），并深耕北欧与低地国家的电影节文化 |
| 月度影讯 | Daily Screening | 分享欧陆银幕上正在放映的专题放映、实验影像事件和艺术院线动态 |
| 专题企划 | Special Project | 邀请各路朋友，共同拆解欧陆放映生态——从电影资料馆的幕后运作，到华语电影的海外传播 |

## 网站特性

- 像素风格开场动画（Canvas 绘制的海浪与陆地退潮效果）
- 像素点阵 Logo（离岸 + OutSea）
- 海浪视频循环背景 + 胶片噪点蒙版
- 海浪环境音效背景音乐
- GSAP + ScrollTrigger 驱动的滚动动画
- 点阵宋体 + Silkscreen 像素字体

## 技术栈

- 原生 HTML / CSS / JavaScript
- [GSAP](https://greensock.com/gsap/) — 动画引擎
- [ScrollTrigger](https://greensock.com/scrolltrigger/) — 滚动触发动画
- Canvas API — 像素开场动画 & 噪点蒙版
